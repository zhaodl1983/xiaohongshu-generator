# 快速启动指南

## 新功能说明

### 1. 违禁词/敏感词检测 ✨
- **位置**：输入框右上角的「🔍 违禁词检测」按钮
- **功能**：一键检测内容中的违禁词，包括：
  - 广告法违禁词（最、第一、100%等）
  - 医疗健康（治疗、疗效等）
  - 金融理财（保本、稳赚等）
  - 虚假宣传（央视推荐、零差评等）
  - 违规引流（加微信、私信等）
  - 低俗违规
  - 小红书特定违禁词

- **使用方法**：
  1. 在输入框中粘贴内容
  2. 点击右上角「🔍 违禁词检测」按钮
  3. 查看弹窗中的检测结果
  4. 根据提示修改违禁词

### 2. 优化后的界面布局 🎨
- **左侧面板**：更简洁，只保留核心输入功能
- **风格选择**：移到右侧顶部，自动换行网格布局，可添加更多风格
- **空间利用**：右侧预览区更宽敞，展示效果更好

## 启动步骤

### 1. 安装依赖
```bash
cd xiaohongshu-generator
pip install -r requirements.txt
```

### 2. 配置 API Key
```bash
# 复制配置文件
cp config.example.py config.py

# 编辑 config.py，填入你的 Gemini API Key
# GEMINI_API_KEY = "your-api-key-here"
```

### 3. 启动应用
```bash
# macOS/Linux
python3 app.py

# Windows
python app.py
```

### 4. 访问应用
打开浏览器访问：http://localhost:5000

## 使用流程

1. **输入内容**
   - 在左侧输入框粘贴长文（500-5000字）
   - 支持 Markdown 格式，会自动提取图片

2. **检测违禁词**（可选）
   - 点击右上角「🔍 违禁词检测」按钮
   - 查看检测结果，修改违禁词

3. **选择风格**
   - 在右侧顶部查看所有风格（自动换行显示）
   - 点击选择你喜欢的风格

4. **预览内容**
   - 点击「✨ 预览内容」按钮
   - AI 会生成文字内容供你编辑

5. **编辑内容**
   - 点击每张卡片的「✏️ 编辑」按钮修改内容
   - 可以上传图片、修改标题、调整要点
   - 点击「🔄 重新生成」可以让 AI 重新生成该卡片

6. **生成图片**
   - 确认内容无误后，点击「🎨 确认并生成图片」
   - 等待 AI 渲染图片

7. **下载图片**
   - 点击单张图片可以预览和下载
   - 点击「📦 批量下载」可以下载所有图片

## 快捷键

- **图片预览**：
  - `ESC` - 关闭预览
  - `←` - 上一张
  - `→` - 下一张

- **编辑模式**：
  - 点击图片插槽后按 `Ctrl+V` 可以粘贴图片

## 注意事项

1. **API Key**：需要有效的 Google Gemini API Key
2. **内容长度**：建议 500-5000 字，太短或太长效果不佳
3. **图片格式**：支持 PNG、JPG、JPEG、WebP，不支持 GIF
4. **违禁词检测**：仅供参考，实际发布前请以平台规则为准

## 常见问题

**Q: 违禁词检测不准确怎么办？**
A: 敏感词库在 `sensitive_words.py` 中，可以根据需要自行添加或删除词汇。

**Q: 如何添加新的风格？**
A: 在右侧风格选择区会自动换行显示所有风格，后续可以通过修改代码添加更多风格模板。

**Q: 生成的图片在哪里？**
A: 图片保存在 `output` 目录下，文件名为 `slide_0.png`、`slide_1.png` 等。

**Q: 可以修改敏感词库吗？**
A: 可以，编辑 `sensitive_words.py` 文件，在对应的分类列表中添加或删除词汇。
